The Hot Key Service can be used to create Keyboard shortcuts for actions on the page. For example, Ctrl+S can be used to invoke the save operation.


This page demonstrates the capabilities of the Hot Key Service


Press I to increment and D to decrement the counter

Press Ctrl+R to reset the counter

{{demoCtrl.counter}}

Try adding the following code snippet to the script.js file to create your own HotKey

    hotKeyService.bindKeyCombination('Ctrl+H', callbackFunc);

    function callbackFunc () {
      vm.showInfoAlert('Alert:', 'Callback handler invoked');
    }

The Hot Key Service supports various key combinations, including 'Shift+D', 'Ctrl+Home', 'Home', 'Ctrl+Enter', etc.
